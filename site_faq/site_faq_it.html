Come effettuo il log-in?

Il sistema può essere configurato per utilizzare diversi sistemi di autenticazione, incluso LDAP, Netware, e SMB. Contatta l'amministratore se hai problemi di autenticazione. Alcune funzionalità sono accessibili solo per alcuni utenti, gli altri riceveranno questo messaggio Non hai i permessi per modificare questo ogetto. Contatta l'amministratore per ulteriori informazioni. Se il sistema è configurato per utilizzare l'autenticazione LDAP, significa che devi utilizzare le stesse credenziali che utilizzi per accedere all'email.

Perchè non posso cancellare/modificare un appuntamento?

Per cancellare o modificare un appuntamento, devi essere loggato come lo stesso utente che ha creato l'appuntamento. Contatta l'amministratore di una delle stanze o la persona che ha creato l'appuntamento inizialmente se vuoi modificare o cancellare l'appuntamento.

Come creo un appuntamento ricorrente?

Cliccando sull'orario desiderato visualizzerai la pagina di prenotazione. Seleziona l'appropiato Tipo ripetizione. La stanza verrà prenotata alla stessa ora, finchè Ripeti fino al, nei giorni determinati dal Tipo ripetizione.

  • Giornaliero ripete la prenotazione tutti i giorni.
  • Settimanale ripete la prenotazione in quei giorni della settimana che hai selezionato in Ripeti giorno. Per esempio, usa la ripetizione settimanale per prenotare la stanza ogni Lunedì, Martedì, Mercoledì e Venerdì; Selezione quei giorni in Ripeti giorno. Se non selezioni alcun giorno in Ripeti giorno, la prenotazione si ripeterà lo stesso giorno di ogni settimana della prenotazione iniziale.
  • Mensile ripete la prenotazione lo stesso giorno ogni mese, per esempio il 15 del mese.
  • Annuale ripete la prenotazione lo stesso mese e giorno, per esempio ogni 15 Marzo.
  • Infine Mensile, giorno corrispondente ripete la prenotazione un giorno ogni mese, lo stesso giorno della settimana nella stessa posizione ordinale del mese. Usa questa ripetizione se per esempio vuoi prenotare il primo Lunedì, il secondo Martedì o il quarto venerdì del mese. Non utilizzare questa ripetizione dopo il 28 del mese.

Come elimino un'istanza di un appuntamento ricorrente?

Seleziona il giorno/stanza/ora che vuoi cancellare e seleziona Cancella.

Come prenoto due stanze contemporaneamente?

Non puoi. Attualmente il sistema non prevede la prenotazione di due stanze contemporaneamente. Devi prenotarle singolarmente. Controlla che l'orario scelto sia disponibile per entrambe le stanze prima di prenotare.

Top

La creazione del mio appuntamento è fallita perchè troppi inserimenti!

Qualsiasi appuntamento non può creare più di 365 inserimenti. E' necessario limitare il numero di appuntamenti creati. Questo numero può essere modificato se necessario.

Cosa accade se più di una persona programma lo stesso appuntamento?

La risposta corta è: La prima persona che clicca sul pulsante Salva vince. Dietro le quinte, il sistema utilizza un database relazionale multi-utente, multi-processo che può gestire centinaia di utenti simultanei.

Che differenza c'è tra Interno ed Esterno

Di default, MRBS definisce due tipi di appuntamento. Interno significa che all'incontro saranno presenti solo dipendenti. Appuntamenti Esterno significa che all'incontro potrebbero partecipare clienti, venditori, investitori ecc. Il sito permette di gestire fino a 10 tipi se necessario. Le prenotazioni sono evidenziate nel calendario principale con un colore corrispondente al tipo, a la leggenda dei colori è visibile a fondopagina.

Come funziona il sistema e chi l'ha scritto?

The Meeting Room Booking System è un software open source che è distribuito sotto licenza Gnu Public License(GPL). Questo significa che il software è gratuito per l'utilizzo, distribuzione e modifiche.

Il sistema è principalmente scritto in PHP, che è una programmazione open source che può essere inserita nelle pagine web e simile come concetto a Microsoft active server pages. PHP è estremamente ottimizzato per gli accessi ai databases.

Il database utilizzato è MySQL o PostgreSQL. MySQL è un database sempre GPL molto veloce, multi-processo, multi-utente e strutturato in SQL (Structured Query Language). PostgreSQL è un database orientato agli ogetti SQL relazionale ed open source.

Il sistema può girare su molteplici piattaforme, inclusi PC con sistema operativo Linux. Linux, è un sistema operativo simil-unix gratuito ed open source.

Il web server è un altro pezzo di software open source e gratuito. L'Apache web server è il più utilizzato al mondo.

Riassumendo: ogni pezzo del sistema, dal sistema operativo all'applicazione è completamente gratuita compresi i sorgenti.
